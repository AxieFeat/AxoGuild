# AxoGuild
Система кланов для Bukkit 1.18.2+

В разработке...

# Особенности:
  - Система уровней;
  - Система клановых войн с аренами;
  - Топы кланов по репутации/убийствам;
  - Кастомизируемые ранги в кланах;
  - Приветствие при вхождении в клан;
  - Создание китов для клановых войн;
  - Банк клана.
